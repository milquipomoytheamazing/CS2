# BudgetBuddy: Smart Expense and Savings Tracker

# CS2
Computer science is the study of computers and computational systems. This is the repository of Milqui Pomoy in CS2.

BudgetBuddy is a simple tool to help users track income, expenses, and savings goals. It provides clear transaction records, visual summaries, and progress updates to make money management easy and motivating.

# Features
Add income and expenses
Categorize spending (food, school, fun, etc.)
Track progress toward savings goals
View spending charts and monthly summaries

# How it Works
Set a savings goal.
Log income and expenses by category and date.
View spending summaries, savings progress, and reports.
